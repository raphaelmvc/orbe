# Orbe

Orbe é um aplicativo gratuito de finanças pessoais para Windows, pensado para uso privado do proprietário e de seus familiares.

O projeto está na fase de implementação orientada pela [especificação de design](docs/superpowers/specs/2026-07-13-orbe-design.md) e pelo [roteiro de implementação](docs/superpowers/plans/2026-07-13-orbe-roadmap.md).

## Direção técnica

- Aplicativo local-first com Tauri, React, TypeScript e SQLite criptografado.
- API Node.js/TypeScript e MySQL hospedados na Hostinger.
- Sincronização privada entre computadores Windows do mesmo usuário.
- Site público somente para apresentação, documentos e download.

## Estado

Design de produto e arquitetura aprovados. A implementação começa pela fundação do monorepo e pelas regras financeiras puras.
